Overview

This extension exposes the OES_texture_half_float functionality to WebGL. There are no WebGL-specific behavioral changes. Consult the above extension for all documentation, issues, and new functions and enumerants.

IDL

module webgl {

interface OES_texture_half_float {
  const GLenum HALF_FLOAT_OES = 0x8D61;
};

};
